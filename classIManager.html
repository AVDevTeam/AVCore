pure virtual

Method for callbacks registration.

Enables plugin to register custom callbacks for supported events.

Parameters
[in]pluginPointer to plugin instance that registers the callback.
[in]callbackIdArbitrary integer value that will server to identify the callback within the plugin. This value will be passed to plugin from IManager when the event occurs.
[in]eventTypeType of event to which the callback will be attached.
[in]priorityInteger value of callback priority. Callbacks with lesser priority will be called first.
Returns
reserved (always 0)
