DECLSPEC_IMPORT UCHAR AVCommIsExcludedPID ( HANDLE  PID)

Implements KM event scanning exclusion based on PID.

This API should be called before submitting events via AVCommSendEvent to check whether the process is exluded (truested). By default AVCore service process is trusted.

Parameters
[in]PIDCurrent PID.
Returns
Excluded (TRUE).
